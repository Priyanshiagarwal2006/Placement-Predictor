# 🚀 Placement Predictor (Machine Learning Project)

## 📌 Overview
The **Placement Predictor** is a Machine Learning-based project designed to predict whether a student will be placed or not based on key academic attributes such as **CGPA and IQ**.

This project demonstrates a complete **end-to-end ML pipeline**, including data preprocessing, analysis, model training, and prediction. It highlights how Machine Learning can be applied to solve real-world problems in the education and recruitment domain.

---

## 🎯 Objectives
- Predict student placement outcomes using ML algorithms
- Analyze the impact of academic performance on placements
- Build a complete ML workflow from raw data to prediction
- Gain practical exposure to data preprocessing and model evaluation

---

## 📊 Dataset Description

The project uses two datasets:

### 🔹 1. placement.csv
- Original raw dataset
- Contains unprocessed student data
- May include inconsistencies or unclean values

### 🔹 2. modified_placement_data.csv
- Cleaned and preprocessed dataset
- Ready for Machine Learning model training
- Data cleaning includes handling missing values and formatting

### 📌 Features Used:
- **CGPA** – Academic performance indicator  
- **IQ** – Intelligence score  
- **Placement Status** – Target variable (Placed / Not Placed)

---

## ⚙️ Machine Learning Pipeline

The project follows a structured ML workflow:

1. **Data Collection**  
2. **Data Cleaning & Preprocessing**  
3. **Exploratory Data Analysis (EDA)**  
4. **Feature Selection**  
5. **Model Training**  
6. **Model Evaluation**  
7. **Prediction**

---

## 🧠 Model Implementation
- Applied Machine Learning algorithms (e.g., Logistic Regression)
- Trained the model on cleaned dataset
- Evaluated performance using standard metrics
